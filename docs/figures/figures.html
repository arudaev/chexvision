Project-report figures

Publication-grade redraws of the four report diagrams — feature maps as tapering tensor volumes, a restrained slate-blue pathway, coral reserved for the primary multi-label task and key events. Inter + JetBrains Mono throughout.

Figure 1End-to-end pipeline. Top lane: the one-time data build and per-run input pipeline. Bottom lane: the per-epoch training loop, from the dual-head forward pass through the combined weighted-BCE loss, gradient accumulation, and best-checkpoint upload to the model hub.
Figure 2Model 1 — custom SE-ResNet. A 7×7 stem downsamples the input; four residual stages (depths 3-4-6-3, widths 64-128-256-512) progressively halve spatial resolution while doubling channels; a global average pool feeds the shared 512-dim vector to the two task heads. Inset: the SE residual block.
Figure 3Model 2 — DenseNet-121. The ImageNet-pretrained dense backbone (dense connectivity, strong feature reuse) feeds a 1024-dim pooled vector into a shared feature layer (Linear 1024→512, ReLU, dropout 0.3), then the same two task heads used by Model 1. Inset: dense connectivity.
Figure 4Two-phase fine-tuning. Phase 1 freezes the backbone and trains only the new heads at lr = 1e-3, letting them stabilize. Phase 2 (from epoch 6) unfreezes the whole network at a 10× lower lr = 1e-4 for gentle end-to-end domain adaptation.
